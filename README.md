# Master's Thesis — Project Structure
## Task-Specific NLP Pipelines vs. LLMs for Cross-Lingual Text Analysis
**University of Trier | Computational Linguistics**

```
/project
├── data/
│   ├── raw/                  # Original CSV corpus — READ ONLY, never modify
│   ├── processed/            # Cleaned corpus, translations, embeddings
│   │   ├── corpus_clean.pkl          # Deduplicated, encoding-fixed DataFrame
│   │   ├── corpus_translated.pkl     # +English translations (cached)
│   │   └── corpus_embeddings.pkl     # Sentence-BERT embeddings (cached)
│   └── samples/              # Stratified validation samples (n=150-200)
│       ├── stratified_sample.csv     # Master sample used across ALL tasks
│       └── sample_metadata.json      # How sample was drawn (for reproducibility)
│
├── notebooks/
│   ├── 00_setup_verification.ipynb   # ← START HERE: verify all tools work
│   ├── 01_corpus_audit.ipynb         # Load, deduplicate, encode-fix, stats
│   ├── 02_preprocessing.ipynb        # Segmentation, metadata, translation
│   ├── 03_stratified_sampling.ipynb  # Draw and lock validation sample
│   ├── 04_ner_pipeline.ipynb         # spaCy / Stanza / Flair NER
│   ├── 05_ner_llm.ipynb              # Mistral-7B NER via prompting
│   ├── 06_ner_comparison.ipynb       # Pipeline vs LLM evaluation
│   ├── 07_topic_pipeline.ipynb       # BERTopic + LDA
│   ├── 08_topic_llm.ipynb            # LLM zero-shot topic labeling
│   └── 09_topic_comparison.ipynb     # Evaluation + temporal analysis
│
├── src/
│   ├── pipeline/
│   │   ├── __init__.py
│   │   ├── preprocessing.py          # Text cleaning, encoding, segmentation
│   │   ├── translation.py            # OPUS-MT wrapper with caching
│   │   ├── ner.py                    # spaCy/Stanza/Flair NER runners
│   │   └── topic_model.py            # BERTopic + LDA runners
│   ├── llm/
│   │   ├── __init__.py
│   │   ├── inference.py              # HuggingFace InferenceClient wrapper
│   │   ├── prompts.py                # Prompt templates (versioned)
│   │   └── consistency.py            # Self-consistency sampling + voting
│   ├── evaluation/
│   │   ├── __init__.py
│   │   ├── metrics.py                # COMET, BERTScore, F1, Kappa wrappers
│   │   ├── agreement.py              # Cross-model agreement, Fleiss' Kappa
│   │   └── bootstrap.py              # Bootstrap confidence intervals
│   └── utils/
│       ├── __init__.py
│       ├── cache.py                  # Disk-cache decorator for expensive ops
│       ├── logging_config.py         # Structured logging setup
│       └── colab_helpers.py          # Drive mount, GPU check, session mgmt
│
├── outputs/
│   ├── ner/                  # NER results, entity lists, comparison tables
│   ├── topics/               # Topic distributions, labels, temporal trends
│   ├── frames/               # Frame detection results (if pursued)
│   └── figures/              # All publication-ready figures
│
├── docs/
│   ├── corpus_statistics.md          # Auto-generated by 01_corpus_audit.ipynb
│   ├── experiment_log.md             # Manual log of experiments and decisions
│   └── prompt_registry.md           # All prompts used, versioned with dates
│
├── requirements.txt          # Pinned dependencies
├── config.yaml               # Global config (paths, seeds, hyperparameters)
├── .env.example              # Template for API tokens (never commit .env)
├── .gitignore
└── README.md
